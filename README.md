# pyratext
Pyratext generates 'pyramid text' from the arguments it is given.

Example input: `Hello World!`

Example output:
```
H
He
Hel
Hell
Hello
Hello W
Hello Wo
Hello Wor
Hello Worl
Hello World
Hello World!
Hello World
Hello Worl
Hello Wor
Hello Wo
Hello W
Hello
Hell
Hel
He
H
```
